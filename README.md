# Galactica-Aeternum / Master Application
Garry's Mod Server that I have mad and the characters (jobs) and their categories.

The entire project was independently completed by Emre Kulaber without any partners.

Put the files into: /garrysmod/addons/darkrpmodification/lua/darkrp_customthings

To customize player models and weapons on your server, you need to modify the 'jobs.lua' file according to the addons installed. It's important to note that these customizations won't work unless tailored specifically, as they depend on the specific workshop items I use. If not customized, the default settings from the existing workshop content will apply.
